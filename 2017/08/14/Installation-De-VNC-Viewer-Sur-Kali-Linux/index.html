Installation De VNC Viewer Sur Kali Linux

Publié par 0patch en Aug 2017
  #Linux #Réseaux
Installation de Logiciels

Salut!

Ce tutoriel consiste a installer VNC Viewer sur Kali Linux. Peut-être marchera pour les autres dérives de Debian, je n’est pas essayé. Comme on dit Am a big fan of Kali Linux . Pour mieux comprendre, vous pouvez lire mon article sur Installation De Serveur VNC Sur Kali Linux - Raspberry Pi. Snip … Allong au coeur du sujet.

Pré-requis:

  • Un ordinateur (Kali Linux)

Prémièrement, télécharger VNC Viewer pour Debian ici

Ensuite ouvrez un terminal et tapez les commande suivantes pour extraire les fichiers .deb du fichier .tar.gz et désinstaller tightvncserver & xtightvncviewer :

Maintenant, nous pouvons installer VNC Viewer. Pour celà:

$ dpkg -i VNC-Server-X.X.X-Linux-x86.deb 
$ dpkg -i VNC-Viewer-X.X.X-Linux-x86.deb

Pour finir, chercher VNC dans la liste d’applications installées, ouvrez et acceptez la licence.

Vous pouvez maintenant utiliser VNC Viewer.
