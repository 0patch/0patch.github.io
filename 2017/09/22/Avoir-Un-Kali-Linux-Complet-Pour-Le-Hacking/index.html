Avoir Un Kali Linux Complet Pour Le Hacking

Publié par 0patch en Sep 2017
  #Linux
Systèmes d'Exploitation

Salut!
Dans mes postes précedent, nous avons installé et configuré Kali Linux. Maintenant Kali Linux utilise aussi plus d’espace, mais toujours est-il que nous n’avons aucun outils installé ou peu je dirai. Dans cet article, nous verrons comment installer tous les outils manquant de Kali Linux, afin d’avoir le nécessaire pour notre voyage dans le monde de la sécurité et de l’investigation numérique.

Pré-requis:

  • Un ordinateur avec VNC Viewer (Linux ou Windows) et Installation De Serveur VNC Sur Kali Linux - Raspberry Pi ou un client SSH comme putty (Windows uniquement)
  • Un Raspberry Pi3 Model B
  • Une carte SD ou mémoire (16 Giga octets minimum).
  • Un câble RJ45 (Ethernet) - (croisés ou non)
  • Un chargeur pour Raspberry Pi Model B ou tout autre chargeur 5V/2A
  • Configurer le partage de connexion Wi-Fi vers Ethernet pour Linux ou Windows

Pour ce faire, connecter vous a Raspberry Pi soit par Secure Shell (SSH) ou en utilisant VNC. Une fois connecté, entrez la commande:

$ apt-get install kali-linux-full

Celà prendra énormement de temps selon votre connexion internet. Une fois installé, vous aurez un système pret pour hacker .
